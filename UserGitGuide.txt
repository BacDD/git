-> keo source tu` git vê` local
--------------------------------------------------------
2. git status
-> biet tinh trang cac file
--------------------------------------------------------
3. git add * or git add BacDang.txt
-> add file vao project
--------------------------------------------------------
4. git status
-> found new file
--------------------------------------------------------
5. git commit -m"add file vao local"
-> add new file to local 
--------------------------------------------------------
6. git push
-> dong bo len github
